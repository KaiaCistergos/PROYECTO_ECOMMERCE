# Proyecto de finalización Módulo N°2

El proyecto consiste en la creación de Front-end de una tienda e-commerce.

## Lenguajes utilizados
- HTML.
- JavaScript.
- CSS.

### Principales secciones de la página

- Cuenta con una barra de navegación.
- La barra de navegación tiene dos links funcionando que redirigen al comprador a otra página siendo "Dona madera" y el icono de "Carrito".
- El carrito tiene un contador de productos añadidos y un botón para vaciar el carrito.
- Cuenta con un carousel de imagenes de madera.
- Cuenta con una sección de productos.
- Cuenta con tarjeta de productos con dos botones funcionales para agregar o eliminar dicho producto.
- Los botones de agregar y eliminar, al ser clickeados, arroja una alerta si se añadió o eliminó un producto.
- Cuenta con un breve footer.
